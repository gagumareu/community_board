#개발 목표
커뮤니티 게시판 웹 개인 프로젝트
-카테고리 별로 게시판 분류
-케테고리 안에 소분류로 원하는 태그 게시판만 분류 가능


#사용기술
서버 : apache - tomcat - 9.0.11

Spring Tool Suite 3 'Mybatis'

Database : Oracle SQL Developer

사용 언어 : Java, HTML, CSS, Jquery, SQL

버전 관리 : Git
